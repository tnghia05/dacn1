# Backend Placeholder

Thu muc nay de san cho backend cua he thong Esport community.

## Goi y cau truc backend (de mo rong sau)

- `src/modules/auth`: Dang nhap, dang ky, phan quyen
- `src/modules/users`: Quan ly thong tin nguoi dung
- `src/modules/posts`: Bai viet, binh luan, thao luan
- `src/modules/reviews`: Danh gia san pham
- `src/modules/trending`: Trend detection
- `src/modules/moderation`: Kiem duyet noi dung
- `src/modules/sentiment`: Tich hop AI sentiment/toxic
